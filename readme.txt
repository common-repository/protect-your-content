== Description ==
Verbiete das Kopieren von Texten und Bildern bzw. füge Dein Copyright hinzu.

### Feature/Settings Overview ###
* Das Markieren der Seiteninhalte unterbinden
* Verbiete CMD und CTRL + A (Alles auswählen)
* Verbiete CMD und CTRL + C (Kopieren)
* Verbiete CMD und CTRL + U (Quellcodeaufruf)
* Verbiete CMD und CTRL + V (Einfügen)
* Erlaube das Kopieren und Einfügen in Textboxen und -feldern

== Frequently Asked Questions ==

Ja, Sie können das Plugin global für alle Seiten und Beiträge aktivieren / deaktivieren.

== Installation ==

= Plugin Installation =

1. Upload 'ProtectYourContent.zip' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings** and configure the **ProtectYourContent.zip**.
